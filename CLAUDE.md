## Project Overview

HikeSmart is a full-stack hiking recommendation system for Hong Kong. It combines GeoJSON trail data, real-time weather from the HK Observatory, AI recommendations via Google Gemini 2.0 Flash, and transport accessibility via Google Maps APIs.
